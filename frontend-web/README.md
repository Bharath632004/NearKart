# NearKart Frontend Web App

Customer-facing web application built with Next.js and React.

## Tech Stack
- Next.js 14 (App Router)
- React.js
- Tailwind CSS
- Axios (API calls)
- Google Maps React SDK
- Razorpay Web SDK

## Key Screens
- Splash / Landing Page
- Login & OTP Verification
- Home (GPS-based nearby shops)
- Search & Categories
- Product Details
- Cart & Checkout
- Payment Gateway
- Real-time Order Tracking
- Order History
- Wishlist & Wallet
- Profile & Address Management

## Run
```bash
npm install
npm run dev
```
