# Admin Service

Platform administration and management backend.

## Features
- Admin Dashboard APIs
- User Management (ban, verify, roles)
- Merchant Approval & Management
- Delivery Partner Management
- Order Oversight & Intervention
- Refund Management
- Coupon & Offer Management
- Audit Logs & Activity Logs
- System Notifications

## Tech
- Java 21, Spring Boot
- PostgreSQL
- Spring Security (Admin RBAC)
