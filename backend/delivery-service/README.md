# Delivery Service

Manages delivery partner operations and real-time order routing.

## Features
- Delivery Partner Registration & KYC
- Vehicle Verification
- Order Assignment (auto + manual)
- Pickup & Delivery OTP Verification
- Live Location Tracking
- Route Optimization (Google Maps)
- Earnings & Wallet Management

## Tech
- Java 21, Spring Boot
- PostgreSQL
- Google Maps Routes API
- Apache Kafka (delivery events)
- WebSocket (live tracking)
