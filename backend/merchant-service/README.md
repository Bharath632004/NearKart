# Merchant Service

Manages merchant/shop registration, KYC, and shop operations.

## Features
- Merchant Registration & KYC
- Shop Profile Management
- Shop Verification Workflow
- GPS-based Shop Location (PostGIS)
- Promotions & Offers
- Sales Reports & Analytics
- Settlement Management

## Tech
- Java 21, Spring Boot
- PostgreSQL + PostGIS
- AWS S3 (KYC document storage)
