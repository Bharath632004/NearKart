# Analytics Service

Business intelligence and reporting for admin, merchants, and platform.

## Features
- Sales Analytics (daily, weekly, monthly)
- Customer Behavior Analytics
- Merchant Performance Reports
- Delivery SLA Reports
- Revenue & Commission Reports
- Real-time Dashboard Data

## Tech
- Java 21, Spring Boot
- PostgreSQL (aggregated views)
- Redis (dashboard cache)
- Apache Kafka (event consumer)
