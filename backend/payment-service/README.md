# Payment Service

Handles all payment transactions for customers and settlements for merchants.

## Features
- Razorpay Integration (UPI, Cards, Netbanking, Wallets)
- Payment Verification & Webhooks
- Refund Processing
- Wallet Transactions
- Merchant Settlement Cycle
- Transaction History

## Tech
- Java 21, Spring Boot
- PostgreSQL
- Razorpay SDK
- Apache Kafka (payment events)
