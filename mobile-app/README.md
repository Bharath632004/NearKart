# NearKart Mobile App

Cross-platform Flutter application for customers and delivery partners (Android & iOS).

## Tech Stack
- Flutter 3.x
- Dart
- Provider / Riverpod (state management)
- Google Maps Flutter
- Firebase Messaging (FCM)
- Razorpay Flutter SDK

## Modules
### Customer App
- GPS detection & nearby shops
- Product browsing & search
- Cart, checkout, payment
- Real-time order tracking
- Wallet & reviews

### Delivery Partner App
- Available orders
- Pickup & navigation
- OTP confirmation
- Earnings & wallet

## Run
```bash
flutter pub get
flutter run
```
