# NearKart Admin Panel

Web-based admin and merchant management dashboard.

## Tech Stack
- React.js
- Tailwind CSS
- Recharts / Chart.js (analytics)
- Axios

## Admin Screens
- Dashboard (platform KPIs)
- User Management
- Merchant Approval & Management
- Delivery Partner Management
- Orders Overview
- Payments & Refunds
- Coupons & Offers
- Analytics Reports
- Settings & Audit Logs

## Merchant Screens
- Merchant Dashboard
- Product & Inventory Management
- Order Acceptance
- Sales Reports
- Settlements

## Run
```bash
npm install
npm start
```
