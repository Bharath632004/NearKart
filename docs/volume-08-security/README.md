# Volume 8 – Security (80+ Pages)

## Security Architecture

- **JWT Authentication**: Access token (15 min) + Refresh token (7 days)
- **OAuth 2.0**: Google Sign-In support
- **RBAC**: Role-Based Access Control (Customer, Merchant, Delivery, Admin)
- **Password Hashing**: BCrypt (cost factor 12)
- **Data Encryption**: AES-256 at rest, TLS 1.3 in transit
- **SQL Injection Prevention**: Parameterized queries, Spring Data JPA
- **XSS Prevention**: Content Security Policy, input sanitization
- **CSRF Protection**: CSRF tokens on state-changing requests
- **Rate Limiting**: Per-IP and per-user rate limiting at API Gateway
- **API Security**: HTTPS only, API key for third-party integrations

## Compliance
- **OWASP Top 10** compliance checklist
- Regular security audits
- Penetration testing plan
