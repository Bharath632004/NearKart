# Volume 5 – API Documentation (250+ Pages)

Base URL: `https://api.nearkart.in/v1`

## API Modules

### Authentication APIs
- `POST /auth/register`
- `POST /auth/login`
- `POST /auth/otp/verify`
- `POST /auth/token/refresh`
- `POST /auth/logout`

### Customer APIs
- Products, Categories, Cart, Orders
- Wishlist, Reviews, Payments, Tracking

### Merchant APIs
- Shop, Products, Inventory, Orders, Reports

### Delivery Partner APIs
- Accept Order, Pickup, Navigation, Deliver, Earnings

### Admin APIs
- Users, Merchants, Orders, Payments, Reports, Analytics

---
Every API includes:
- Endpoint & Method
- Request Headers & Body
- Response Schema
- Status Codes
- Validation Rules
- Error Messages
- Authentication Requirements
- Sample JSON Request/Response
