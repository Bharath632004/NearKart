# Volume 10 – Testing (100+ Pages)

## Testing Strategy

| Type | Tool | Scope |
|---|---|---|
| Unit Testing | JUnit 5, Mockito | Individual service methods |
| Integration Testing | Spring Boot Test | Service-to-DB interactions |
| API Testing | Postman, REST Assured | All API endpoints |
| UI Testing | Selenium, Cypress | Frontend web flows |
| Mobile Testing | Flutter Driver | Flutter app flows |
| System Testing | End-to-end scenarios | Full order lifecycle |
| Load Testing | Apache JMeter | 1000+ concurrent users |
| Stress Testing | Gatling | Breaking point analysis |
| Security Testing | OWASP ZAP | Vulnerability scanning |
| UAT | Manual testers | Business scenario validation |
| Performance Testing | k6 | API response time benchmarks |

## Test Coverage Goal
- Backend services: ≥ 80% code coverage
- Critical paths (payment, order): ≥ 95% coverage
